kakumeiteki

An AI coding assistant that runs entirely on your own Mac.
Un asistent AI de programare care rulează complet pe Mac-ul tău.

free / gratuit no account / fără cont works offline / merge offline nothing is uploaded / nu se trimite nimic

EN Before you start you need:
  • A Mac with Apple Silicon (M1, M2, M3, M4) — check: Apple menu → About This Mac
  • macOS 14 (Sonoma) or newer · 8 GB RAM minimum (16 GB = noticeably faster)
  • ~6 GB free disk space
  • Internet — only for the install. Afterwards it works offline forever.
  • About 20 minutes. Everything is free.
RO Înainte să începi ai nevoie de:
  • Un Mac cu Apple Silicon (M1, M2, M3, M4) — verifică: meniul Apple → Despre acest Mac
  • macOS 14 (Sonoma) sau mai nou · minim 8 GB RAM (16 GB = vizibil mai rapid)
  • ~6 GB spațiu liber pe disc
  • Internet — doar pentru instalare. După aceea merge offline pentru totdeauna.
  • Aproximativ 20 de minute. Totul este gratuit.

EN

So why use it, when cloud AI is faster?

  • Nothing leaves your machine. No account, no API key, no subscription, no company reading your files.
  • It works with no internet — plane, train, bad wifi, anywhere.
  • It costs nothing, ever. No billing per question.
  • It cannot lie about its work — every result that changed a file ends with a line the program computes itself, not the AI.
  • It is genuinely good at the boring jobs: cleaning, duplicates, renaming, small fixes.
RO

Atunci de ce să îl folosești, când AI-ul din cloud e mai rapid?

  • Nimic nu pleacă de pe calculatorul tău. Fără cont, fără cheie API, fără abonament, fără vreo firmă care să îți citească fișierele.
  • Merge fără internet — în avion, în tren, cu wifi prost, oriunde.
  • Nu costă nimic, niciodată. Fără plată la întrebare.
  • Nu poate minți despre ce a făcut — orice rezultat care a modificat un fișier se termină cu o linie calculată de program, nu de AI.
  • Este cu adevărat bun la treburile plictisitoare: curățenie, duplicate, redenumiri, corecții mici.

Is it safe? / Este sigur?

EN
  • It is locked inside the folder you start it in. It cannot read or write anywhere else on your Mac.
  • It asks before every single file change — you see the before/after and answer y (yes) or n (no).
  • Everything is undoable. Type kaku undo and the file goes back exactly as it was. Deletions go to a recovery store, never straight to rm.
  • Dangerous commands are refused — sudo, pipe-to-shell and 30+ other classes are blocked in every mode.
  • Your secrets stay secret — .env files, keys and passwords are refused and hidden automatically.
  • Every action is written to a log you can read.
  • It cannot fake a result. A line like — verified 1/1 · exit 0 is written by the program. If nothing proved the change, it says UNVERIFIED instead of pretending.
RO
  • Este închis în folderul din care îl pornești. Nu poate citi sau scrie nicăieri altundeva pe Mac.
  • Te întreabă înainte de fiecare modificare de fișier — vezi înainte/după și răspunzi y (da) sau n (nu).
  • Totul se poate anula. Scrie kaku undo și fișierul revine exact cum era. Ștergerile merg într-un depozit de recuperare, niciodată direct în rm.
  • Comenzile periculoase sunt refuzate — sudo, pipe-to-shell și peste 30 de alte categorii sunt blocate în orice mod.
  • Secretele rămân secrete — fișierele .env, cheile și parolele sunt refuzate și ascunse automat.
  • Fiecare acțiune este scrisă într-un jurnal pe care îl poți citi.
  • Nu poate falsifica un rezultat. O linie ca — verified 1/1 · exit 0 este scrisă de program. Dacă nimic nu a dovedit modificarea, scrie UNVERIFIED în loc să se prefacă.
